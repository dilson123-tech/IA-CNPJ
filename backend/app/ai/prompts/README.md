# Prompts (IA-CNPJ)

Este diretório guarda prompts/versionamento para providers reais (ex.: OpenAI).

Regras:
- Nada aqui deve quebrar o modo determinístico.
- Providers reais só rodam quando `AI_ENABLED=true`.
- Prompts devem ser pequenos, auditáveis e versionados.
